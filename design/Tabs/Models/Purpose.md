Purpose: The technical engine room where users manage their AI provider connections and model preferences.





API Key Management: Secure fields for OpenAI, Anthropic, and Google Gemini.



Lock/Unlock States: Visual indicators (lock icons) show if a key is active and authenticated. Keys are masked for security.



Model Selector: A toggle-based list of available LLMs (e.g., GPT-4o, Claude 3.5 Sonnet) allowing users to enable/disable specific models for their workspace.



Custom Providers: A dedicated section at the bottom for advanced users to add their own API endpoints (e.g., local Ollama instances or private enterprise models).



Usage Stats: The Right Sidebar provides real-time telemetry on token consumption and average latency across providers.
