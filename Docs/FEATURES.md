# Features

## Core Features

### Prompt Analysis
AI detects:

Intent
Category
Missing parameters
Optimization opportunities

### Adaptive Quiz
AI asks:

Audience
Output format
Detail level
Constraints
Reasoning depth

Questions adapt to prompt type.

### Optimization Studio

Right panel contains:

Optimization depth
Reasoning control
Output verification
Structure enforcement
Example generation

### Structured Prompt Output

Output includes:

Goal
Audience
Format
Constraints
Quality level
Verification instructions

Final optimized prompt generated after structure creation.

### Multipass Optimization

Pass 1 Analysis

Pass 2 Structure build

Pass 3 Optimization

Pass 4 Verification

### Prompt History

Stores:

Original prompt
Parameters
Optimized prompt
Versions

### Reoptimization

Users can modify parameters and regenerate optimized prompt.

### Collapsed Quiz Memory

Answered questions collapse but remain visible.
